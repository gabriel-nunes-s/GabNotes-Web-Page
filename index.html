GabNotes

What is GabNotes?

GabNotes is a digital note-taking application that allows users to create, manage, and organize their notes, tasks, and to-do lists

Technologies in use

GabNotes was developed using a combination of technologies including JavaFX for the user interface, Java for the application logic, and potentially libraries like ControlsFX and DraggableFX for specific functionalities like drag-and-drop interactions.
